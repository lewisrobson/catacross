# catacross
concatenate two files across, combining them one line at a time
Requirements: lines be the same length (the right side file can be padded out with new lines)

for example

file_left
-line 1: hello
-line 2: an

file_right
-line 1: world
-line 2: example

combined_file
-line 1: hello world
-line 2: an example
